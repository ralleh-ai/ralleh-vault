# GUIDELINES.md — VAULT Quality Rules

## Source of Truth Rules
- `raw/` is immutable evidence.
- `wiki/` holds durable canonical knowledge.
- Canonical claims require explicit `sources`.
- Relationships must be represented with meaningful wikilinks.

## Quality Bar
A VAULT output is production-grade only when:
- frontmatter is complete and valid
- uncertainty is stated, not hidden
- links are resolved and useful
- decision/procedure gates are compliant
- strict doctor passes on the affected root
