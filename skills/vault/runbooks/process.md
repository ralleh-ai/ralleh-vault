# Runbook: vault.process (VAULT-FAST)

1. Enumerate `vault/Inbox/` and newly landed `vault/raw/` refs.
2. Deduplicate and remove obvious noise.
3. Classify target note type.
4. Apply matching template from `templates/`.
5. Fill baseline frontmatter and key bullets.
6. Add candidate `related` links.
7. Escalate non-routine judgment items to VAULT.
8. Empty Inbox for processed items.
