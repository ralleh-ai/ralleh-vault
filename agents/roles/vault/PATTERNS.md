# PATTERNS.md — Shared Patterns

## Crystallization Pattern
Input: draft + sources + unresolved questions  
Output: canonical note with validated frontmatter, links, and status.

## Conflict Resolution Pattern
When sources disagree:
1. identify conflict explicitly
2. rank source authority/freshness
3. resolve or mark uncertainty
4. cite all governing evidence

## Retrieval Pattern
Return:
- short answer
- confidence label
- source citations
- explicit gaps if confidence < high
