# Runbook: vault.crystallize (VAULT)

1. Gather all source references and prior related notes.
2. Resolve conflicts; preserve uncertainty explicitly.
3. Write canonical note in `vault/wiki/<Type>/`.
4. Set frontmatter status and confidence.
5. Add bidirectional links and update nearby MOCs.
6. Update `vault/wiki/index.md` and append `vault/wiki/log.md`.
7. Trigger approval gate for major Decisions/Procedures when required.
