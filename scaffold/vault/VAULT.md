# VAULT.md

Welcome to Ralleh Vault.

## Purpose

A local-first Markdown knowledge vault for durable company intelligence.

## Rules

- `raw/` is immutable source capture.
- Durable knowledge goes in `wiki/`.
- Use wikilinks to connect knowledge.
- Keep provenance explicit in frontmatter `sources`.

## Workflow

Capture → Process (VAULT-FAST) → Crystallize (VAULT)
